## 误区二：CN2 GT 就是 CN2 GIA，买哪个都一样

**真相：CN2 GT 是"半程优化"，国内段照样走 163（AS4134）。**

CN2 GT 的 GT 是 Global Transit 的缩写。它的路由逻辑是：国际段走 CN2 网络（AS4809），但数据进入国内之后，就回落到 163 骨干网（AS4134）。用 traceroute 检测的特征是：路径前半段看到 `59.43.x.x`（这是 CN2 的 IP 段），后半段又变成 `202.97.x.x`（163 段）。

用个比喻：你打了高速公路出发，快到目的地了临时下高速走国道，颠簸几十公里再到家。

搬瓦工对应 CN2 GT 线路的套餐是 **KVM 系列**（默认洛杉矶 DC2 机房，可切换 DC3 CN2 机房，但 DC3 走的是 CN2 GT，流量仅为三分之一）。价格便宜，入门用没问题，但别对晚高峰速度有太高期待。
